## v1.0.28

*2026-03-13*

### initial public release

- full task orchestration platform with multi-agent support (claude code, cursor, codex)
- real-time pty terminal streaming via xterm.js in web dashboard
- skill pipeline system with confirmation, choice, and input stages
- multi-machine worker fleet with intelligent task routing
- lark bot integration with interactive message cards
- developer cli (`ov`) for task management and terminal attachment
- operations cli (`overlord`) for server/worker lifecycle management
- role-based access control (developer, lead, admin)
- totp two-factor authentication
- personal access tokens for api/cli authentication
- cursor remote tunnel support for workspace access
- project-level configuration with pipeline definitions
- audit logging for all administrative actions
- cloudflare zero trust deployment support
- sqlite database with wal mode
- redis-backed task queue via bullmq
- comprehensive web dashboard with dark/light theme
